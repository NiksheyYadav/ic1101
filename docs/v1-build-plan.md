# Aetheris Studio v1 Build Plan

## Milestones

### M1: Platform Foundations (Weeks 1-4)
- Auth, workspace, project services
- API gateway and service skeletons
- Metadata database schema

### M2: Data + Pipeline (Weeks 5-8)
- Dataset ingestion and versioning
- Schema validation and profiling
- Preprocessing pipeline block engine (MVP blocks)

### M3: Training + Realtime (Weeks 9-14)
- Job orchestration and resource planner
- Checkpointing + pause/resume/cancel
- Metrics streaming and live dashboard

### M4: Experiment + Registry + Deploy (Weeks 15-18)
- Run comparison engine
- Model registry lifecycle
- Managed endpoint deployment

### M5: Hardening + Beta (Weeks 19-22)
- RBAC polish
- Alerting/notification pipeline
- Load tests, chaos tests, UX polish

## Team Structure (Suggested)
- 1 Product Lead
- 1 UX Lead + 2 Product Designers
- 1 Staff Architect
- 2 Backend squads (4-5 engineers each)
- 1 Frontend squad (4 engineers)
- 1 MLOps squad (4 engineers)
- 1 SRE (shared)
- 2 QA/SET

## Launch Checklist
- Security review complete
- Reliability SLO dashboards live
- Disaster recovery runbook tested
- Onboarding tutorials and docs published
- Billing metering validated
