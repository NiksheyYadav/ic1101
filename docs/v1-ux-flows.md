# Aetheris Studio v1 UX Flows

## Primary Flow: Dataset to Deployed Model
1. Create Project
2. Import Dataset(s)
3. Validate Schema
4. Configure Preprocessing Pipeline
5. Select Task + Model
6. Configure Hyperparameters
7. Configure Compute
8. Start Training
9. Monitor Realtime Dashboard
10. Evaluate + Compare
11. Register Best Model
12. Deploy API Endpoint

## Beginner Mode
- Wizard-first journey
- Fewer hyperparameters surfaced
- Guided defaults and explanations
- Inline tips and warnings

## Expert Mode
- Full hyperparameter matrix
- Advanced scheduler/optimizer controls
- Resource topology controls
- Deep telemetry and logs

## Realtime Monitor Layout (v1)
- Header: run status, controls, ETA
- Left: training/validation metric charts
- Right: system telemetry + anomaly cards
- Bottom: logs + checkpoint timeline

## Empty State Patterns
- Datasets page: "Upload your first dataset" CTA
- Training page: "No runs yet. Start with recommended baseline"
- Registry page: "Promote a run to register your first model"
